## Piper TTS — voice customisation limits (discussed with Miguel)

- Piper voices are fixed pre-trained models — no pitch/tone/emotion knobs
- `speed` parameter supported (0.5–2.0) via openedai-speech's OpenAI-compatible API
- No voice cloning — pre-trained voices only
- Upgrade path for more control: Kokoro (82M, local, slightly more expressive), Qwen3-TTS (voice cloning from 3s audio, needs GPU), ElevenLabs (gold standard, full style controls)
- For telephony use (8kHz), Piper quality differences are negligible vs ElevenLabs
- For Slack voice messages, ElevenLabs Brian is noticeably better
