### Confluence Tool Schema Note

`confluence_upload_attachment` requires `content_id` (not `page_id`) and `file_path`. Does NOT accept `filename` or `mime_type` as separate args. Correct call:
```
confluence_upload_attachment(content_id="<id>", file_path="<path>", comment="optional")
```

`confluence_update_page` does NOT accept a `version` argument. Correct call:
```
confluence_update_page(page_id, title, content, content_format)
```
